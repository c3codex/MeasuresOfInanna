# THE MEASURES OF INANNA  
### Seven Gates of Surrender · Nine Epithets of Reclamation  
### An Oracle Codex by c3codex

This repository houses the full codex architecture, artwork, text, oracle cards, metadata, and NFT production files for **The Measures of Inanna** — a three-material ceremonial system built in Obsidian, Crystal, and Marble.

## Structure

### 0 — Front Matter  
Dedications, Invocation, Epigraph, Usage Guide, Epilogue, and Blessings.

### 1 — Obsidian Gates  
Seven Gates of Surrender, each with:
- Artwork  
- Gate Text (ritual, breath, descent function)  
- Mandala (optional)

### 2 — Crystal Epithets  
Nine Epithets of Reclamation, each with:
- Artwork  
- Epithet Text (meaning, breath pattern)  
- Mandala (optional)

### 3 — Marble Temple  
Structural ceremonial pages:
- Cards of Invocation, Dedication, Epigraph, Blessing, Epilogue, and Closure

### 4 — Oracle Cards  
Three suits:
- Obsidian (Gates)
- Crystal (Epithets)
- Marble (Temple cards)

### 5 — Art Assets  
All static artwork files categorized by material.

### 6 — Metadata  
NFT JSON files for minting via Zora or alternate platforms.

### 7 — Animations  
Optional animated layers for obsidian, crystal, and marble textures.

---

## Purpose
This codex is built to function simultaneously as:

1. A **sacred manuscript**  
2. An **oracle deck**  
3. A **multi-material NFT collection**  
4. A **breath-activated ritual system**  
5. A **Codex extension within the c3 ecosystem**
